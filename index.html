Projects

Missing Value Imputation for Tabular Data

Apr. 2022 - Apr. 2024

Design a deep learning model to uncover the inherent relationships within incomplete tabular data for imputation, such as the similarity between samples and the correlations between variables. Works include:

  • Propose a tracking-removed autoencoder model combined with a fuzzy framework for missing value imputation. (Published in Information Sciences, IF=8.1, first author)
  • Propose a deep learning architecture for collaborative optimization of fuzzy clustering and intra-class attribute regression for missing value imputation. (Published in Neural Computing and Applications, IF=4.5, Student first author)
  • Propose a multi-task learning model combining graph neural networks and tracking-removed autoencoders to solve the cumulative errors introduced by the "imputation before classification" in the data classification task. (Published in Applied Intelligence, IF=3.4, Student first author)

Adaptive Learning-based Time Series Recovery

Nov. 2023 - Feb. 2025

Propose a dynamic graph neural network model to address the issues caused by incomplete multivariate time series, including high-accuracy missing value imputation, prediction models resistant to missing values, and a lightweight dynamic graph neural network for imputation. Core Works:

  • Propose a dynamic graph convolutional network-based imputation model, which combines the advantages of recurrent neural networks and graph neural networks to achieve advanced imputation performance. (Published in Neural Networks, IF=6.0, Student first author)
  • Propose a dynamic graph attention network-based prediction model, which employs only the available values for prediction, addressing the propagation errors in the traditional 'impute-then-predict' approaches. (Under review in IEEE Trans. Systems, Man, and Cybernetics, IF=8.6)
  • Propose a lightweight dynamic graph convolutional network for missing value imputation, which significantly reduces memory consumption and processes more time steps compared to existing advanced graph neural networks. (Under review in IEEE Trans. Instrumentation & Measurement, IF=5.6)

Diffusion-based Scalable Video Coding

Feb. 2024 - Oct. 2024

Design a real-time communication system that adaptively adjusts the resources required for video compression and restoration based on the network environment, ensuring stable and efficient communication.

  • Design a three-layer video encoding architecture: The first layer performs semantic information down sampling, using a diffusion-based super-resolution method in the decoder to enhance video quality. The second layer discards frames with minimal motion and applies adaptive compression to preserve high-quality video. The third layer allocates additional resources to deliver the highest quality video. (Published in IEEE IoT Journal, IF=10.6, Co-first author)
